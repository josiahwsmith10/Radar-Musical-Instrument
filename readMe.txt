Folder Name		|	Notes
fcnn			|	Holds pretrained FCNNs for image enhancement in .mat files 
data			|	Saved captures
include			|	DCA1000EVM interface codes and necessary libraries are stored here, as well as calibration coefficients in a .mat file and system objects
music			|	The system object and functions for the Radar Musical Instrument
saved			|	Saved radar parameters are stored here in .mat files and can be loaded into the GUI on the Config page
utility 		|	Functions used by the GUI
